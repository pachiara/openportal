Description
Editor plugin Bootstrap Bootstrap allows you to add elements within the content editor of the system. Bootstrap elements facilitate the work of editing the contents.

Configuration

After activating the plugin, select the "Content / Post / Add New" , in the form of Post editor, in the menu on the toolbar of the editor find the link Format display the menu and select any of the menu options Bootstrap format.

To add an item to bootstrap, first we opened a post and look for the "Insert Template" button, clicking on this button will open a mode where you can choose any item as shown in the image on the left column.
